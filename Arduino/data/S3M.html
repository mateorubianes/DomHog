Sobre la compañía

Somos el grupo 14 de sexto año 2022.
Este proyecto se llama DomHog y se basa en el control remoto de una casa por WIFI mediante el uso de una página web. Sus integrantes son Santiago Andrés Aguilar Vieito, Martín Ariel García, Máximo Poratti Iturzaeta y Mateo Joaquín Rubianes.

Página Web

La forma principal de control y manejo de DomHog es mediante un sitio web desarrollado por nosotros en donde se muestra cada dispositivo conectado y su estado, además de poder ver los parametros sensados como lo son la temperatura y la presión atmosférica.

Módulos utilizados

El proyecto cuenta con dos tipos de módulos principales, la UCP y la UCR.
La UCP es la unidad de control de procesos, que es la parte central del proyecto y la que controla cualquier periférico que se pueda agregar. En esta se podrá ver la temperatura, presión atmosférica y además se puede acceder a cada UCR para ver o configurar su estado.
Por otro lado estan las UCR, o unidades de control remoto. Estas son los periféricos que, en esta presentación son un enchufe controlado por temperatura, una luz dimmerizable con 10 valores posibles, una luz de encendido y apagado que se maneja tanto por la página web como por una ficha física, y una persiana controlable.
